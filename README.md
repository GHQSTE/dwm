# Clown's build of [dwm](https://dwm.suckless.org/)

> Keybindings in this build is customized for <a href="https://colemakmods.github.io/mod-dh/">Colemak Mod-DH</a> keyboard layout.
> Visit config.def.h/config.h to change keybindings to your likings.

## Patches applied in order

1. [xrdb](https://dwm.suckless.org/patches/xrdb/)
2. [scratchpads](https://dwm.suckless.org/patches/scratchpads/) &
[shiftview](https://github.com/LukeSmithxyz/dwm/blob/master/shiftview.c)
3. [fullgaps](https://dwm.suckless.org/patches/fullgaps/)
4. [restartsig](https://dwm.suckless.org/patches/restartsig/)
5. [hide vacant tags](https://dwm.suckless.org/patches/hide_vacant_tags/)
6. [actualfullscreen](https://dwm.suckless.org/patches/actualfullscreen/)
7. [movestack](https://dwm.suckless.org/patches/movestack/)
8. [focusonnetactive](https://dwm.suckless.org/patches/focusonnetactive/)
9. [alwayscenter](https://dwm.suckless.org/patches/alwayscenter/)
